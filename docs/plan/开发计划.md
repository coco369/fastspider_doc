# 功能开发计划

**fastspider**是开箱即用的一个高性能爬虫框架, 其打造的是一个轻量级开发爬虫, 上手简单。 以及配套有可视化爬虫监控的运维系统。最终实现爬虫生命周期的维护与状态监控。
